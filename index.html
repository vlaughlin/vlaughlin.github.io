About Me

Hello! My name is Valerie! I am a front-end web developer based in Central Florida. I specialize in designing and developing custom websites.

I recently got certified as a Front-End Web Developer at Treehouse. I like to learn the latest trends in web development and apply those skills to my projects.

I got interested in web development at the early age of 13, and now I want to pursue web development as a freelancer to help people's ideas come to reality with my technical skills.

Services Offered

I have skills in HTML5, CSS3 and JavaScript. Technology changes constantly, so I like to keep myself updated with the latest web-development trends.

I have skills in building responsive websites that adjust to fit mobile phones, tablets, and desktop. I know web accessibility, which allow anyone to use the web regardless of physical or hardware differences.

I know to style and optimizing SVG graphics. I know frameworks such as Bootstrap, JQuery, SASS and React. Contact me for your next project!
